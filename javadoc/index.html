About Flux Capacitor Client

Flux Capacitor provides messaging-as-a-service for distributed applications.

Project Modules

This project has declared the following modules:

Name Description
io.flux-capacitor:common Library with common components for Flux Capacitor clients and service.
io.flux-capacitor:java-client Default Java client library for interfacing with Flux Capacitor.
io.flux-capacitor:test-server In memory server that can be used for testing applications in your local environment.
io.flux-capacitor:proxy Server library that forwards web requests to Flux Capacitor.
